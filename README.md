# Relatório com Parâmetros e Storytelling

## Sobre o Projeto
Este repositório contém a resolução do desafio de implementação de parâmetros de campos no Power BI. O objetivo deste projeto é aprimorar um relatório existente, adicionando dinamismo às análises e criando uma narrativa de dados (storytelling) estruturada para apresentação ao cliente.

## Diretrizes Atendidas

O painel foi desenvolvido seguindo estritamente as instruções propostas:

- **Primeira Visão (Parâmetros de Categorias):** Implementação de um visual dinâmico focado na análise através da alternância de diferentes categorias.
- **Segunda Visão (Parâmetros de Valores):** Construção de um visual que utiliza parâmetros de campos baseados em métricas numéricas, permitindo a transição de visualização entre valores como Vendas (Sales) e Lucro (Profit).
- **Consistência Visual:** Toda a estilização aplicada nos novos visuais e na página segue rigorosamente o padrão visual estabelecido no relatório do módulo anterior.
- **Storytelling para o Cliente:** Criação de uma página dedicada e formatada especificamente para apresentação. Os visuais foram organizados de maneira lógica para contar uma história clara sobre os dados, sem apenas jogá-los na tela.
